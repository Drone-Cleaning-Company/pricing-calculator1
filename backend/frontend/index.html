1. Introduction

Welcome to the Window Cleaning Pricing Calculator! This tool is designed to streamline the process of estimating costs for window cleaning projects, manage calculations, and provide insights for business operations.

2. Registration

To get started with the Pricing Calculator, follow these steps:

  1. Users: During registration, specify your country to receive country-specific content and currency. Users do not need to put Admin Key.
  2. Admins: Enter the provided admin key during registration to gain admin privileges. The Admin Key is iAmTheAdmin.

3. Saving Costs

IMPORTANT: Before using the calculator, ensure you have saved your operational costs and other costs in the system by navigating to "Operational Cost" and "Other Costs" and clicking "Save Configuration". This will help ensure accurate pricing calculations.

4. User Guide

For All Users:

  1. Window Cleaning Page:
    • Enter the client's name and address.
    • Select the number of buildings.
    • Input the dimensions of the buildings.
    • Edit glass coverage if necessary.
    • Click "Calculate sq footage".
    • Click "Continue to Pricing Details".
  2. Pricing Details Page:
    • Select the building height category.
    • Enter the number of floors.
    • Choose the cleaning frequency.
    • Apply discounts if applicable (percentage or fixed amount).
    • Click "Calculate Final Price".
  3. Saving Calculations:
    • After reviewing the details, click "Save Calculation".
  4. Viewing Saved Calculations:
    • Access the dashboard and click "Saved Calculations".

For Admins:

  • Accessing P&L Page:
    • IMPORTANT: Always save your current calculation before navigating to the P&L page.
  • Managing Calculations:
    • View and delete saved calculations as needed.
  • User Management:
    • View and delete user accounts.

5. Important Notes

  • Users can only view calculations for their registered country.
  • Admins have access to calculations from all countries.
  • Always save your "Operational Cost" and "Other Cost" after your first login.
  • Always save your calculation before navigating away from the pricing page, especially when accessing the P&L page.

6. Features

  • Detailed window cleaning cost estimation.
  • Discount application (percentage or fixed amount).
  • Savings calculations management.
  • Country-specific view for users.
  • Comprehensive admin controls including P&L analysis and user management.
